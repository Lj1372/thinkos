ThinkOS Sprint 1 — Design Spec

Date: 2026-04-16  ·  Scope: Full UI rebuild + 10 features  ·  Target: index.html + app.py
UI Rebuild Blind Spot Detector Full Council Session Save & History Voice Input Templates Modal Dark/Light Mode Keyboard Shortcuts Mobile Bottom Nav 2 New API Endpoints
ThinkOS currently has 4 tools in a tab layout. Sprint 1 transforms it into an OS-feeling app with a persistent sidebar, a new Full Council mode, saved history, and 6 UX upgrades. Everything stays in a single HTML file + Flask backend — no build toolchain, no frameworks.

1 Layout — Sidebar Architecture

Left sidebar (240px, fixed)

  1. Logo — "ThinkOS" with purple accent
  2. Current Thought textarea (auto-grows, ~80px min) + 🎙️ voice button inline
  3. ⚡ Full Council button — primary CTA, purple
  4. Tools nav — REI Council, The Ladder, Kingdom Lens, Socratic, 🔍 Blind Spot
  5. 📋 Templates — ghost button, opens modal
  6. Recent Sessions — last 8, clickable to restore
  7. 🌓 Theme toggle — bottom of sidebar

Right main area

  • Shows active tool output — no tabs
  • Thought shown as read-only pill at top with 🔒 + "edit" link
  • Tool switching via sidebar nav only

Mobile (< 640px)

  • Sidebar collapses → fixed bottom nav bar (5 tool icons)
  • Thought field → sticky collapsible bar at top of main
  • Full Council → floating action button (bottom-right)

2 Global "Current Thought" Field

  • Single source of truth — feeds all tools when you switch
  • Persists to localStorage across refresh
  • 🎙️ voice button: Web Speech API, appends to existing text
  • First-visit hint: "feeds all tools"

3 Full Council Session

Runs: REI → Ladder → Kingdom Lens → Blind Spot (Socratic excluded — conversational)

Report layout

  1. Synthesis card (top) — AI reads all 4 results, writes 3–4 sentence synthesis + one sharp question. Gradient bg: dark purple → dark teal.
  2. Full detail sections (collapsible) — REI, Ladder, Kingdom, Blind Spot each rendered in full
  3. Bottom — Export as Markdown + Share buttons

New endpoint

POST /api/synthesis
Body:    { rei, ladder, kingdom, blind_spot }
Returns: { synthesis, synthesis_question }

4 Workflow Templates

📋 Templates button → modal overlay (max 560px, 2-column grid of template cards)

TemplateToolsFor
🔮 Big Life DecisionREI → Ladder → Kingdom → Blind SpotMajor choices
⚡ Conflict with SomeoneREI → SocraticRelationship tension

Click template → modal closes, thought field focused. If thought exists → runs immediately.

5 Save & History

  • localStorage key: thinkos_sessions — max 50 sessions
  • Each session: id, title (first 40 chars of thought), timestamp, all tool results
  • Sidebar shows last 8 — click to restore instantly (no API call)
  • Hover → delete button (×)
  • "View all" → sessions modal with search
  • Export → clean .md file download from Full Council report

6 Blind Spot Detector

New tool: "What perspective is completely missing here?"

POST /api/blindspot
Body:    { situation, context? }
Returns: { missing_perspective, why_its_missing, reframe, blind_spot_question }
  • Forced to ONE specific missing angle — no generic advice
  • Cross-references REI results if available
  • UI: 3 cards (Missing / Why / Reframe) + blind spot question

7 Premium Progress Bar

  • Full-width animated gradient bar at top of main area (replaces spinner)
  • Colour matches active tool
  • Animates 0% → 85% during call, snaps to 100% on completion, fades out
  • Subtle glow effect
  • Loading messages still cycle underneath (existing system)

8 Dark / Light Mode

  • Default: dark. Toggle: 🌓 at bottom of sidebar.
  • Detects prefers-color-scheme on first load
  • Persisted to localStorage key thinkos_theme
  • CSS swap: data-theme="light" on <html>, all colours already CSS variables

9 Keyboard Shortcuts

ShortcutAction
Cmd/Ctrl + EnterSubmit current tool
Cmd/Ctrl + KFocus thought field
Cmd/Ctrl + Shift + FTrigger Full Council
Cmd/Ctrl + 1–5Switch tools
EscapeClose any modal

10 Mobile Bottom Nav

  • Fixed 60px bar, full width, 5 tool icon buttons
  • Active tool highlighted in tool colour
  • Full Council → floating action button (bottom-right, purple)
  • Templates → ⋯ menu

Success Criteria

  • Sidebar layout works on desktop and mobile
  • Global thought field feeds all tools automatically
  • Full Council runs all 4 tools and shows synthesis + full detail
  • Templates modal opens, selecting one pre-fills and runs
  • Sessions saved to localStorage, restorable from sidebar
  • Blind Spot Detector returns unique, specific insight
  • Progress bar replaces spinner, feels premium
  • Dark/light toggle persists across refresh
  • All keyboard shortcuts work
  • Mobile bottom nav replaces sidebar on small screens
  • Export to Markdown works
